# color-mixing-lamp
an Arduino based lamp that mixes colors using different sensor input. 
this can for example be used with a termometer where hot= red and cold= blue


the first project is code for making an color changing lamp using photo transistors that detect red green and blue light. the data from those sensors get sent to the RGB led which then tries to mimic the color of the light in your room 

the second project is code for making an temprature changing light based on the same work, only that this time the sensor that is used to determine the color of the RGB led is how hot and cold it is in the room.
